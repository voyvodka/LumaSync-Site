## At a glance

| | LumaSync | Hyperion |
|---|---|---|
| Platform | macOS (primary), Windows, Linux | Linux (primary), macOS / Windows community ports |
| Deployment | Desktop app with tray UI | Headless service + separate UI |
| Hue bridge | Native DTLS 1.2 PSK | Via forwarder plugin |
| USB LEDs | Adalight (CH340, FT232) | Adalight + many others |
| Setup | Install and pair | Compile, configure, run as service |
