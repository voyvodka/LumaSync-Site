## Prerequisites

- **Operating system**: macOS 13+, Windows 10+, or a mainstream Linux desktop (GNOME, KDE, Hyprland).
- **Hardware**: at least one of the following is required for the app to actually light anything up:
  - A Philips Hue Bridge (gen 2 or newer) with a configured Entertainment Area.
  - A WS2812B LED strip connected to a CH340 (`1A86:7523`) or FT232 (`0403:6001`) USB controller running Adalight-compatible firmware.
